Клонирование проэкта с github  

```commandline
git clone <сылка на репезитории>
```
Добавление файлов в наш коммит
```commandline
git add . 
```

Отмена добавления 
```commandline
git reset 
```
Проверка статуса
```commandline
git status 
```
Создание комита с его название 
```commandline
git commit -m "Название"
```
Сохрание нашего комита в репозитории 

```commandline
git push
```
